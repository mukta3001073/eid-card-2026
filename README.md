# # Eid Wish Card 🌙

A fun and interactive Eid greeting card web app where users can send wishes and explore secret duas.
---
## Live Demo
Check out the live website here: [Live Demo](https://eid-card-2026.netlify.app/)

---
## Features
- 🌟 **Eid Wish Wall** – Users can leave their Eid wishes.  
- 🙏 **Secret Dua Button** – Surprise dua message for users.  
- 📱 **Responsive Design** – Works on desktop and mobile.  
- 💻 Built with HTML, CSS, and JavaScript.  
- Celebrate eid 
- Counting eid days time

## Extra / Future Features
- 🔗 **Share Wishes** – Users can share their wishes on WhatsApp, Messenger, or Facebook.  
- 🎉 **Animated Eid Cards** – Confetti or glow effects for celebration.  
- 🕌 **Random Dua Generator** – Secret Dua button shows a random dua each time.  
- 🌗 **Dark Mode / Light Mode Toggle** – Switch UI for user preference.  
- 💾 **Save Wishes Locally** – Browser local storage to keep wishes even after reload.  
- ⏳ **Countdown to Eid** – Displays remaining time until Eid.  
- ✨ **Instant Card Preview** – See the card preview while typing a wish.

## How to Use
1. Open the [Live Demo](https://eid-card-2026.netlify.app/) link.  
2. Enter your name and Eid wish in the **Eid Wish Wall**.  
3. Click the **Secret Dua Button** to receive a surprise dua message.
